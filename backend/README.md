# 🔧 DevTrack Backend

Node.js backend that exposes system metrics using shell scripts.

---

## 🚀 Run locally

```
npm install
node server.js
```

---

## 🐳 Docker

```
docker build -t devtrack .
docker run -p 3000:3000 devtrack
```

---

## 🌐 API Endpoints

- `/cpu`
- `/ram`
- `/disk`

---

## ⚙️ Deployment

Deployed on Render.
